== Description ==

Create feature rich troubleshooting guides for your website visitors to follow

== Installation ==

1. Unzip the contents of `wpdevhub-troubleshooting-guides.zip`
2. Upload the full `wpdevhub-troubleshooting-guides` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Access the Troubleshooting Guides plugin through the links on the left of WordPress administration panel.

== Screenshots ==

1. The WPDevHub Troubleshooting Guides plugin allow you to present step-by-step instructional content in an organized fashion that is easy to read for your users.
2. The plugin is built directly into the WordPress post editor system as a custom post type.  This allows content to be used as a standalone post.
3. Basic editor functionality has been extended allowing you to easily add as many troubleshooting steps as needed.
4. Final content is then provided to readers in an organized fashion that looks natural with standard WordPress designs and templates.
5. The plugin also uses the built in WordPress taxonomy tools to create custom Guide Categories as desired.
6. Additional settings are provided to help control and extend the capabilities of the plugin.
